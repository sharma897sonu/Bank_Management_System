# Project Overview

The Bank Management System is designed to automate basic banking activities and maintain customer records securely. It reduces manual work and provides a structured way to handle banking transactions using a database-driven approach.

## Technologies Used

- Programming Language: Java (Core Java)
- Database: MySQL
- JDBC: Java Database Connectivity
- IDE:IntelliJ IDEA 
- Operating System: Windows

## Features

- User Authentication (Login System)
- Create New Bank Account
- View Account Details
- Deposit Money
- withdraw Money
- Balance Inquiry
- transaction History
- Secure Database Storage

## ATM Facility
- ATM Card Number & PIN Authentication
- Cash Withdrawal via ATM
- Balance Inquiry via ATM
- Mini Statement (Last Transactions)
- Daily Withdrawal Limit Handling
- Secure PIN Validation

## How to Run the Project

1. Clone the repository: ```bash
   git clone https://github.com/sharma897sonu/Bank_Management_Systemt ```
2. Open the project in your preferred Java IDE.
3. Create a database in MySQL:
   CREATE DATABASE bank_management;
4. Import the SQL file provided in the project (if available).

5. Update database credentials in the Java JDBC configuration file:
   String url = ```bash jdbc:mysql://localhost:3306/bank_management ```;
   String username = "root";
   String password = "your_password";
6. Run the main Java file.
